## How to use it

You can just use this template after clone it.

### 1) Clone

> git clone https://github.com/greatasher7/react-typescript-redux.git

### 2) Git initialize (You run terminal at root of this project)

> rm -rf ./.git

> git init

### 3) Install

> npm install

### 4) Run dev-server

> npm start
